Compiler Design and implementation

I implemented a compiler employed nanopass framework that compiled the source code with a lisp-like syntax to x86 assembly. The features I implemented included arithmetic, integer and boolean variables, conditions, loops, arrays, tuples, functions, garbage collection, and type checking. I also implemented several optimizations, which included partial evaluation on algebraic and conditional for simplifying the abstract syntax tree, constant propagation and removal of dead variable assignment using dataflow analysis, as well as elimination of “jump” instructions.
